Este conjunto de scripts para MATLAB contém uma interface para estudo do manipulador Kuka KR3 Agilus.

Encontra-se as funções:

Kr3fk : Cinemática direta.
Kr3ik : Cinemática inversa.
Kr3init : Inicialização.
Kr3fj : Relação direta de velocidades.
Kr3ij : Relação inversa de velocidades.
Kr3plot : Gráfico do manipulador mediante parâmetros.
kr3ptp : Movimento ponto a ponto.
KukaKR3 : Uma classe que recebe os atributos do manipulador.

A instrução, assim como exemplo de utilização, encontra-se no script "Example.m". Ali será possível entender o funcionamento das ferramentas.
